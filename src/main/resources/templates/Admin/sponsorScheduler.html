How it works:

  • Processes one sponsor every 30 seconds (3 Buffer API calls per sponsor)
  • Calls Gemini AI to generate a short version for Twitter/Bluesky (max 280 chars)
  • LinkedIn gets the full sponsor description
  • Posts are scheduled once per day at midday (Paris time), starting tomorrow
  • Automatically stops when all sponsors are processed
  • Skips sponsors from previous years and those without description
